# twitter_bot_pi

## Usage

clonw and copy

```
% cd
% git clone git@github.com:funnythingz/twitter_bot_pi.git
% cd twitter_bot_pi
% cp config.rb.sample config.rb
```

edit config

```
% vim config.rb
```

bundle

```
% bundle install
```

add cron

```
% whenever --update-cron
```
